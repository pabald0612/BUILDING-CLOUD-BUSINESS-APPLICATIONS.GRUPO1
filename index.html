Arquitectura del Proyecto Grupo ABBCA

Capa de Extracción

Procesos ETL para extraer datos de Excel, sistemas de escritorio y Power Apps.

⬇️

Capa de Almacenamiento de Datos

Base de datos Azure SQL para centralizar y organizar la información.

⬇️

Capa de Aplicación

Interfaz de usuario con Power Apps para registrar y gestionar datos.

Pantalla de inicio: punto de acceso a todas las funcionalidades de la app.

Buscar registro existente: permite localizar leads por Nro Fiscal o Nombre y Apellido.

Agregar registro: formulario inicial para nuevos leads o prospectos.

Registro de lead: captura datos completos del prospecto (nombre, contacto, Nro Fiscal, correo, celular).

Vista general del Nuevo Lead Registrado y Revisión de Departamentos Disponibles.

Detalle de un departamento: características, ubicación, precio e imágenes.

Envío de correo automático para consultar detalles de un departamento.

Acción para iniciar el proceso de reserva del departamento seleccionado.

Confirmación de reserva enviada automáticamente al correo del usuario.

Muestra información del departamento reservado y estado de la reserva.

Opción para cancelar una reserva existente.

Pantalla de confirmación para ejecutar la cancelación de la reserva.

Mensaje de éxito notificando que la reserva fue realizada correctamente.

Pantalla para realizar la compra definitiva del departamento reservado.

Correo de confirmación automática tras completar la compra.

Pantalla final confirmando la cancelación de la reserva.

Confirmación de compra enviada al correo del cliente.

⬇️

Capa de Datos

Tablas normalizadas: Ventas, Clientes, Sedes, Modelo, Buyback y Método de Pago.

⬇️

Capa de Presentación

Dashboards interactivos en Power BI y reportes en Excel para toma de decisiones.
