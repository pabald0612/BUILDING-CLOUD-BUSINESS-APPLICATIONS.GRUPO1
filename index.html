Arquitectura del Proyecto iStore

Data Sources

Excel y sistemas de escritorio, registros de Power Apps.

⬇️

Data Layer

Base de datos Azure SQL con tablas: Ventas, Clientes, Sedes, Modelo, Buyback, Método de Pago.

⬇️

ETL Layer

Pipelines y Dataflows de Microsoft Fabric + Notebooks Python para automatización y notificaciones.

⬇️

Data Warehouse Layer

Fabric Data Warehouse, medidas calculadas, relaciones y tabla calendario para análisis.

⬇️

Application Layer

Power Apps: Pantallas de Inicio, Ventas, Clientes, Sedes, Buyback, Método de Pago.

⬇️

Presentation Layer

Power BI y Excel: reportes interactivos y dashboards ejecutivos.

⬇️

User Access

Roles y permisos en Azure, notificaciones automáticas por correo.
